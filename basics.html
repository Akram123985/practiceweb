What is Digital Literacy?

Digital literacy goes beyond just knowing how to use a computer. It includes:

  • Understanding different types of digital devices (smartphones, tablets, laptops).

  • Navigating software applications for work, education, and personal use.

  • Communicating online effectively and safely.

  • Evaluating information for accuracy and reliability.

Key Digital Skills You Need to Know

1. Basic Computer and Internet Navigation
  • How to use different operating systems (Windows, macOS, Linux).

  • Understanding internet browsers (Chrome, Firefox, Edge, Safari).

  • Using search engines like Google effectively (Boolean search, advanced filters).

2. Effective Online Communication
  • Writing professional emails (structure, etiquette, avoiding spam).

  • Using video conferencing tools like Zoom, Microsoft Teams, Google Meet.

  • Understanding online collaboration tools (Google Drive, Slack, Trello).

3. Digital Citizenship & Ethical Use of Technology
  • Respecting intellectual property (copyright laws, fair use policies).

  • Understanding online etiquette (Netiquette).

  • Preventing cyberbullying and harassment.

4. File Management and Cloud Storage
  • Organizing files effectively on a computer.

  • Using cloud storage platforms like Google Drive, OneDrive, and Dropbox.

  • Sharing files securely.

5. Keyboard Shortcuts for Efficiency

Windows:
  • Ctrl + C → Copy

  • Ctrl + V → Paste

  • Ctrl + Z → Undo

Mac:
  • Command + C → Copy

  • Command + V → Paste

  • Command + Z → Undo
